# Node.js Console Calculator 

Console calculator that takes an input string containing a mathematical expression and prints the result of its calculation to the console

## Initialization

``` 
npm i
```
## Start

```
npm run start
```
## Testing

```
npm run test
```

## Example of input value

```
1 + 3 - 5 / 5 + ( 7 - 3 ) * 2.5 + ( 3 - 7 )
```
